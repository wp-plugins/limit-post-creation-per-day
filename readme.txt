Plugin that allow you limit number of posts per day each user can create

== Description ==

With that plugin you can set how many post per day each type of user can create. For example you can limit authors to eight post per day, contributors to five posts per day, etc. If some user exceed his limits, he will not able to create news posts.
The Plugin has others options, it allows you set no limits for special users just entering they IDs on the plugin options page.

== Installation ==

Follow the below steps to intall the plugin:
1- Send the folder limit-post-creation-per-day to plugins path, it should be /wp-content/plugins/
2- Acess the blog panel and activate the plugin Limit Post Creation Per Day
3- Now, in the Settings box, a new link will be shown "Limit Post Creation Per Day" Click on it to go to pluin options page.
4- Enter the number of posts each type of users can create per day and click on the Update button. Enter -1 to no limits.
5- In the field "Special Users IDs" you can enter Users IDs separated with comma to remove their post limis
This is all you need to do.
